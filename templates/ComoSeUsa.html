Como se usa?

Para usar el asistente de voz, debes pulsar el boton de hablar. Permite el microfono para que la web pueda reconocer tu voz. Una voz hecho todo esto, podemos usar algunos comando para saludar como: "Hola". Debes hablar alto y claro para que pueda entender bien.
Si queremos hacer busquedas en internet usamos el comando: Buscar en Google y luego decimos la palabra o la frase que queremos buscar.
Si queremos reproducir una cancion usamos el comando: Pon la cancion.. Seguido del tema o el artista que queremos. Puedes decir, pon la radio y se abrira una pestaña con una radio. Si quieres poner una alarma di: Pon una alarma a las 16 38. Si quieres saber el tiempo de hoy di: Que tiempo hace hoy. Para que te cuente un chiste puedes decir: Cuéntame un chiste. Para saber todos los comandos que has usado di: Mostrar historial, y si lo quieres borrar solo di: Borrar historial. Para usar el traductor di: Traduce del español ... Seguido de lo que quieres que traduzca, y a la inversa.
